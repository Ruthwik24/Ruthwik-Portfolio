House Price Prediction Using Scikit-Learn

June 2023 - August 2023

House Price Prediction using Scikit-Learn is a comprehensive machine learning application that analyzes various property features like location, square footage, room counts, and market trends to build accurate valuation models. The project follows a complete data science pipeline - collecting and cleaning real estate data, performing exploratory analysis to identify key price drivers, engineering relevant features, and training multiple regression models including Linear Regression, Decision Trees, and ensemble methods. The system employs rigorous validation techniques and evaluates performance using metrics like MAE and R-squared to ensure reliable predictions. This implementation demonstrates how Scikit-Learn's powerful algorithms can transform raw property data into actionable insights, helping buyers, sellers, and real estate professionals make informed, data-driven decisions about property valuations in dynamic housing markets. The scalable solution handles everything from data preprocessing to model deployment, showcasing practical machine learning applications in the real estate domain.

A Machine Learning Model for Emotion-Based Suggestion and Quote Recommendation

March 2024 - June 2024

This project presents an emotion detection and recommendation system that analyzes facial expressions using a Convolutional Neural Network (CNN) trained on the FER-2013 dataset (containing 48x48 grayscale images of seven emotions). The detected emotion triggers a personalized recommendation system that draws from a custom MQS dataset of 1,680 curated suggestions and motivational quotes (240 per emotion category). The CNN architecture employs convolutional layers with ReLU activation, batch normalization, and dropout layers to prevent overfitting, optimized using Adam with learning rate scheduling. After preprocessing input images through normalization and augmentation techniques, the system outputs both the detected emotion and corresponding recommendations. The complete pipeline - from real-time facial expression analysis to context-aware quote/suggestion generation - provides an interactive tool for emotional awareness and positive reinforcement, with potential applications in mental health support and human-computer interaction systems.

Fine-Tuning ZFNet for Image Classification on the ImageNet Dataset

October 2024 - November 2024

This project implements a fine-tuned ZFNet architecture for image classification on the ImageNet dataset, leveraging transfer learning to optimize performance. Building upon AlexNet's foundation, ZFNet enhances feature extraction through strategic modifications like smaller filter sizes and adjusted strides. The methodology involves freezing initial convolutional layers while fine-tuning only the final fully connected layers, significantly reducing computational costs without compromising accuracy (achieving 96.04% on a 1,000-class subset). A robust preprocessing pipeline standardizes inputs via resizing, cropping, and normalization, while training employs SGD with momentum and Cross-Entropy Loss for efficient optimization. Evaluated against benchmarks like AlexNet and VGGNet, the model demonstrates superior accuracy and generalization (validation loss: 0.47), validated through metrics including precision, recall, and F1-score. This approach balances performance and efficiency, making it viable for real-world applications requiring high-throughput image classification.

Drug–Target Interaction Predictions with Multi-View Similarity Network Fusion

January 2025 - Present

This project presents an advanced deep learning-based framework for predicting drug-target interactions (DTIs), a critical task in drug discovery and repurposing. It integrates multi-view biological data, including drug-drug interactions, protein-protein networks, chemical structures, drug-disease associations, and side effects, to provide a comprehensive understanding of molecular interactions. The system leverages Graph Convolutional Networks (GCNs) to capture complex topological features from molecular graphs and Multi-Layer Perceptrons (MLPs) for high-level feature representation. To enhance prediction accuracy, it incorporates attention mechanisms, including self-attention for drugs and targets, as well as cross-attention between drug-target pairs, enabling the model to focus on the most biologically relevant features. The framework outputs interpretable interaction likelihood scores, providing researchers with actionable insights for decision-making. Built with scalability in mind, the system efficiently processes large-scale biomedical datasets, making it suitable for real-world pharmaceutical research and accelerating the discovery of novel drug-target relationships.
